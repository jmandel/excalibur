# Fixtures

## Generated fixtures

Files under `fixtures/generated/` are generated by `scripts/make_minimal_epubs.py` and are intentionally tiny synthetic test books created for this project.

They are original test data for this repository and may be used for converter regression tests.

Current generated fixtures:

- `minimal.epub`: EPUB3 package with NCX, nav, two XHTML spine files, internal links.
- `css-image.epub`: CSS stylesheet, image resource, two XHTML spine files.
- `svg.epub`: external SVG resource referenced from XHTML.

## Third-party fixtures

Third-party public-domain/open-license samples should be added under `fixtures/third_party/` with source URL and license recorded here before committing.
